
# vk-playground

A basic c++ framework with custom CMake gems to play with Vulkan.

### Dependencies

- CMake 2.8.12+
- [Vulkan SDK](https://vulkan.lunarg.com/),
- [GLFW 3.2](https://github.com/glfw/glfw/tree/3.2),
- [GLM 9.8](https://github.com/g-truc/glm/releases/tag/0.9.8.0).
